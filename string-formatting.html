String Formatting

String format is an unique and useful feature. Let's see how it works:

        print("Hello,{}".format('User'))
        > Hello,User

We use .format operation to replace all '{}' to our word, in this case - User.

"Why do we need this?" - May you ask. Just look at this snippet:

        a = input('Your name: ')
        print('hello, {}'.format(a))
        > Your name: Alexander
        > hello, Alexander

We can make an reusable and compact script!

Also, we can define where should variables go.

        a = 1
        b = 2
        c = 3
        print('{2}, {1}, {3}'.format(a,b,c))
        print('{1}, {3}, {2}'.format(a,b,c))
        > 2, 1, 3
        > 1, 3, 2
