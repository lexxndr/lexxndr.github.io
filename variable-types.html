Variable types.

1.String

String is one of the easiest types. It's not something special, but it has some difficulties. We are gonna review it later.

2.Int(eger)

Integer is one of the easiest variable types, like string. It does not contains any difficulties, but we'll learn it later in future lessons.

3.Float/double

Float is not that easy. Float, with a double is using a floating point numbers with (,) or (.)

        a = 1.34
        print(a)
        > 1.34
    
        a = 1.3
        b = 2.7
        print(a+b)
        print(int(a+b))
        > 4.0
        > 4

You can do any operation with it as with int, but now with floating numbers.

4.Boolean

Boolean is very simple type. It can have only two values: True or False (or, 1 and 0)

Boolean is a logical type, with two of values being only logical.

        a = True
        b = False
        if a == b:
            print('Not gonna print this')
        else:
            print('Gonna print this')
        > Gonna print this
