Python has a print() command. It commands Python to print an argument or string (or any other variables).

        print("Hello World!")
        > Hello World!
        print('Hello World!')
        > Hello World!

As you can see, i made two strings - with ("") and with ('') and, this worked! So, you can use one of those.

Let's continue.

You can put argument in print() command like this snippet: 

        a = 1234
        print(a)
        > 1234

        a = 1234
        print("a")
        > a

If you cover an argument with ("") or (''), its gonna print the name of argument (string)
