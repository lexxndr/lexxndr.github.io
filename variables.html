Variables

As we already saw, python has variables like strings, int, etc.

To define a variable, you need to give a name, and a value

        variable1 = "Hello!"
        variable2 = 123
        print(variable1)
        print(variable2)
        > Hello!
        > 123

There is many types of variables, there's list of them (not full):

String, integer, boolean, float, double.

All of them will be reviewed in other lessons.
