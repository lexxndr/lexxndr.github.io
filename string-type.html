String Variable Type

String is commonly used type of variables, which uses quotes (double or single, "", '') to make string.

        a = "Hello"
        b = "World!"
        print(a)
        print(b)
        > Hello
        > World!

You can use some simple operations, as concantenation (+):

        a = "hello"
        b = " world"
        print(a+b)
        > hello world
