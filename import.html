Import Command

        import *module name*

It's not highly used in first steps of programming, but you will need to import modules.

First, you need to install module. In your CMD, type:

        pip install *module name*

Then, you can import this module in code:

        import numpy

This code snippet will import numpy module.

        import numpy as np

This code snippet will import numpy module as 'np', so, it will be used in the code like that:

        import numpy as np
        np.function()

You can also import certain things with "from" argument:

        from numpy import function
        from numpy import *

This code will import numpy's "function" in your code (first line), and import all numpy's functions in your code (second line)
