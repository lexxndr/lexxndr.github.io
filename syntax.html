Python syntax

Python has simple syntax, but with some difficulties, as:

1.Tabulation

        i=0
        while i <= 10:
        print('hi')
        i++
        print('end')

"What's wrong?" - May you ask. You expect this to repeat n times and after this it should be printing "end", so?

- Not that easy. This code is gonna:

1. Call an error "SyntaxError"
2. Print hi AND end N times.

"How to fix it?" - Just add some tabs.

        i = 0
        while i <= 10:
            print('hi')
            i++
        print('end')

This pseudocode is gonna work.

        > hihihihihihihihihihi
        > end

Tabs is frequently 4 spaces and it helps interpreter to understand what to do.

2. Symbols: ",)({}: , etc

You saw in previous snippet, how i printed "hi" and "end".
To make string, you need to cover your text in quotation marks. Example:

        variable = "Hello World!"

Also, in previous snippet you saw a colon after "10" and it means this:

        while i less than 10 do

The word "do" in this snippet is just a colon.

        while i <= 10:

Also, you can use "[]" for lists, "{}" for dictionaries and "()" for tuples.

        my_list = [1,3,4]
        my_dict = {1:'a', 2:'b', 3:'c'}
        my_tuple = (1,2,3)

        print(my_list[0])
        print(my_dict[1])
        print(my_tuple[0])

        > 1
        > a
        > 1
